# 🌐 Lumora Web

The frontend application for Lumora — a Stellar-powered creative marketplace where artists showcase their work, connect with clients, and receive secure blockchain payments.

## ✨ Features

* Artist portfolio management
* Creative marketplace
* Artwork discovery
* Client dashboards
* Commission requests
* Wallet authentication
* Stellar payments
* Real-time messaging

## 🛠 Tech Stack

* Next.js
* TypeScript
* TailwindCSS
* React Query
* Zustand
* Stellar Wallet SDK
* Soroban Client SDK

## 🚀 Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Application

```bash
npm run build
```

### Start Production Server

```bash
npm start
```
